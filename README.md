## JavaScript Coding Quiz

# Description
 This coding quiz project simulates a typical coding assessment, incorporating multiple-choice questions and interactive challenges to help you practice JavaScript fundamentals. The quiz runs in the browser, featuring dynamically updated HTML and CSS powered by the JavaScript code you'll write. The interface is designed to be clean, polished, and responsive.

# User Story
As a coding boot camp student, I want to take a timed quiz on JavaScript fundamentals that stores my scores. This allows me to gauge my progress compared to my peers.


# Features
Interactive timer to measure the time taken to complete the quiz.
Multiple-choice questions to test JavaScript fundamentals.
Dynamic presentation of questions and answers.

# Technologies Used
HTML
CSS
JavaScript

Run the application to generate quiz https://snunez1231.github.io/JS-Fundamentals-Quiz/

Answer each question to the best of your ability.
